# Tech Stack

## Core
- **React 19**: Frontend framework.
- **TypeScript**: Type safety.
- **Vite**: Build tool and dev server.

## State Management & Data Fetching
- **Zustand**: Client-side state management.
- **TanStack Query (React Query) v5**: Server state management and caching.
- **React Hook Form**: Form handling.
- **Zod**: Schema validation.

## Backend & Database
- **Supabase**: Backend as a Service (BaaS) for Auth, Database (PostgreSQL), and likely Storage.

## UI & Styling
- **Tailwind CSS**: Utility-first CSS framework.
- **Lucide React**: Icon library.
- **Framer Motion / Motion**: Animations.
- **Recharts**: Data visualization.
- **Sonner**: Toast notifications.

## Infrastructure
- **Vercel**: Likely deployment platform (based on `vercel.json`).

## Testing
- **Vitest**: Unit and integration testing.
- **Playwright**: End-to-end testing.
- **React Testing Library**: Component testing.
