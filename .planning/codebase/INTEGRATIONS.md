# Integrations

## External Services
- **Supabase**: Primary database and authentication provider.
- **Dropbox**: Integrated via `dropbox` and `react-dropbox-chooser` for file management.
- **Google Generative AI**: Integrated via `@google/genai` for AI features (e.g., "Study Buddy").

## File Processing
- **Mammoth**: For converting `.docx` files to HTML.
- **PDF.js**: For rendering and processing PDF documents.
- **Marked**: For parsing Markdown content.

## Monitoring & Auditing
- **AuditService**: Internal service for logging system activities (likely stored in Supabase).
- **ActivityMonitor**: Service for tracking user session activity.
